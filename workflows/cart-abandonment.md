# Cart Abandonment Workflow

## 🎯 Objective
Recover users who added products to cart but did not complete purchase.

## 🔁 Workflow Steps

1. User adds product to cart  
2. Wait for 2 hours  
3. If no purchase → send reminder  
4. If still inactive → send follow-up after 24 hours  
5. Exit workflow on purchase  


## 🧠 Learning

- Event-based triggers  
- Time delay automation  
- Behavioural targeting
